[Live demo](https://okidokitokiloki.github.io/rock-paper-scissors/)

## General info
This is a classic game of Rock, Paper, Scissors. It is (currently) played in the console (frontend version otw).

For simplicity, user options are: 
    (case insensitive)
    "r" - rock
    "p" - paper 
    "s" - scissors
